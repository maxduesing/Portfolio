# Portfolio
Personal Project Portfolio

### 🔹 Offline LLM Deployment
Local deployment of a large language model in a fully offline environment.

- **Focus:** Privacy, offline compute, and local system resource management
- **What I did:** configured and ran an open-source LLM locally, managed model files and dependencies, and verified offline functionality
- **Tech:** Python, local inference tools, macOS/Linux
- **Takeaway:** gained hands-on experience with local AI tooling, model constraints, and system-level considerations for offline workloads

### 🔹 Android APK Deployment via BlueStacks

- **Focus:** Application deployment and emulator-based testing
- **What I did:** transferred and installed APK files manually, validated application behavior in an emulated Android environment, and troubleshot compatibility issues
- **Tech:** BlueStacks, Android APKs
- **Takeaway:** developed familiarity with Android app packaging and emulator-based testing workflows

- **Focus:** Networking, DNS, and remote connectivity

- **Tech:** Pi-hole, DNS, Tailscale, Linux
- **Takeaway:** gained practical experience with networking concepts, DNS resolution, and secure remote access

- **Tech:** Linux, BIOS/UEFI, boot loaders
